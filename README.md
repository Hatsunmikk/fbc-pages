# fbc-pages

# Free Books

A mobile-first book discovery and borrowing interface built using HTML, CSS, and JavaScript as part of the Fictional Book Club front-end assignment.

The application allows users to browse books, filter and sort listings, and view detailed information for individual books through a dedicated details page.

# Features

- Home Page
- Mobile-first responsive layout
- Sticky header
- Promotional banner
- Functional filters:
  Fiction / Non-Fiction
  Genre
  Rating (4+)
  Sorting:
  A-Z
  Z-A
  Highest Rated
- Dynamic book grid rendered from - JavaScript data
- Bottom navigation
- Desktop-only warning screen

# Book Details Page

- Dynamic routing using URL parameters
- Book information loaded from a shared dataset
- Hero section with cover image and metadata
- Dynamic background gradient generated from the book cover
- About, Author, Publisher, ISBN, Review Summary, Tags, and Library sections
- Publisher logos rendered dynamically
- Fixed Borrow button
- Sticky bottom navigation

# Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Fonts
- ImageKit-hosted book covers
